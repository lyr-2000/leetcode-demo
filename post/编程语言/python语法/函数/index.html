函数的文档注释

  • 开发中希望给函数添加注释，注释应该定义在函数的下方，用三对引号括起来
  • 三对引号之间编写行说明文字
  • 函数调用位置 使用快捷键，ctrl j ctrl j ,就可以查看函数说明信息

pycache原理

python会把 python模块 编译为 pyc文件，导入的时候，就会进行编译，再次执行同一个模块的话，效率就会提高

变量类型

  • python 分为数字型和非数字型

  • 数值型

    • 整形，int
    • 浮点 ，float
    • 布尔，bool
    • 复数 ，complex

  • 非数字型

    • 字符串
    • 列表
    • 元组
    • 字典

  • 所有非数字型变量都支持下面的特点：

    • 都是 序列，sequence
    • 取值 []
    • 遍历 for in
    • 计算长度，最大，最小值，比较、删除
    • 链接 + ，和重复 *
    • 切片 [:]

列表的操作

终端输入 ipython3, 输入 name = [] ,name. 和 tab键， ipython就会自动显示出这个列表有哪些方法可以调用，比如 append,count,reverse,sort 等方法

删除列表的方法：

  1. li.remove(obj)
