atomic.Value 参考

参考博客

Go的sync/atomic包提供了原子操作，支持的数据类型包括：

int32, int64, uint32, uint64, uintptr, unsafe.Pointer
1
若需要扩大原子操作的适用范围，可以使用atomic包中的Value。利用它可以实现对任意值进行原子得存储与加载。

//值类型
func atomic_value(a uint32) {
	var v atomic.Value
	v.Store(a)
	a = 666
	fmt.Println(a)
	fmt.Println(v.Load())
}

// 引用类型
func atomic_slice(s []uint32) {
	var v atomic.Value
	v.Store(s)
	s[0] = 666
	fmt.Println(s)
	fmt.Println(v.Load())
}
