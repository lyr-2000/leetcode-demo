给定 100
个整数，请你找出其中最大的数字，以及它的输入位置（位置从 1
开始）。

输入格式
共 100
行，每行包含一个整数。

输出格式
第一行输出最大的数字。

第二行输出该数字的输入位置。

数据范围
1≤输入数字≤50000
,
保证输入数字互不相同。

package main
import (
    "fmt"
    "math"
)
func main() {
    var x int
    var ret = int(math.MinInt64)
    var id = 0
    for i:=0;i<100;i++ {
        fmt.Scanf("%d",&x)
        if ret < x {
            ret = x
            id = i
        }
    }
    fmt.Println(ret)
    fmt.Println(id+1)
}
