闭包延时绑定

  • 闭包里面使用闭包外的参数，其值最终调用时候确定下来的。

闭包很强大，但是要小心延时绑定, 一般 可以在代码块前面 重新定义 i ，例如 i:=i

func Delay() {
    fns := make([]func(),0,10)
    for i:=0;i<10;i++ {
        fns = append(fns,func() {
            fmt.Println(i)
        })
    }
    for _,v := range fns { v() }
}
