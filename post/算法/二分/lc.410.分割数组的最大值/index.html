410. 分割数组的最大值

Description

Difficulty: 困难

Related Topics: 贪心 , 数组 , 二分查找 , 动态规划

给定一个非负整数数组 nums 和一个整数 m ，你需要将这个数组分成 m个非空的连续子数组。

设计一个算法使得这 m个子数组各自和的最大值最小。

示例 1：

输入：nums = [7,2,5,10,8], m = 2
输出：18
解释：
一共有四种方法将 nums 分割为 2 个子数组。 
其中最好的方式是将其分为 [7,2,5] 和 [10,8] 。
因为此时这两个子数组各自的和的最大值为18，在所有情况中最小。

  • 1 <= nums.length <= 1000
  • 0 <= nums[i] <= 106
  • 1 <= m <= min(50, nums.length)

class Solution {
public:
    int splitArray(vector<int>& nums, int m) {
        vector<long long> sum(nums.size()+1);
        int n = nums.size();
        for(int i=1;i<=n;i++) sum[i] += (sum[i-1] + nums[i-1]);
        vector<vector<long long>> dp(n+1,vector<long long>(m+1,INT_MAX));     
        dp[0][0] = 0;
        for(int i=1;i<=n;i++) {
            //dp[i][1] = sum[i];// 前i 个 1组
            // 前 i个 一共 m组
            for(int j=1;j<=min(i,m);j++) {
                for(int k=0;k<i;k++) {
                    // max(不分配， 分配(i-k))
                    dp[i][j] = min(dp[i][j],max(dp[k][j-1], sum[i] - sum[k]));

                }
            }
        }
        
        return dp[n][m];
    }

   
};

二分法+贪心

答案？为什么是 <=T

nums = [7,2,5,10,8], 一共有四种方法将nums分割为2个子数组

[7,| 2,5,10,8], max=25
[7, 2 | ,5,10,8], max=23
[7, 2 ,5 |,10,8], max=18
[7, 2 ,5 ,10|,8], max=24

我们可以考虑让 nums 分为m个子数组，最大一组的和 = T, – 只有 18,23,24.25 合法

如果每组的和 <=T, 17以前都不合法，18之后合法， 25 之前合法。

我们的目的就是二分枚举出分割数组的最大值。

注意： 前提条件是非负的数组 ，可以 令 l=0,r = sum(array),枚举中间的值。

class Solution {
public:
    int splitArray(vector<int>& nums, int m) {
        int n = nums.size();
        // int  sum = 0;
        int l=0,r = 0;
        for (auto v: nums) {
            r += v; //上界
            l = max(l,v); //下界
        }

        while(l<r) {
            int T = l+(r-l)/2;
            if (valid(nums,T,m)) {
                r = T; //找到一个上界，继续往下移动
            }else {
                l = T+1;//not valid 
            }
        }
        return l;
    }

    bool valid(vector<int> &nums,int T,int m) {
        int group_sum=0,group_cnt = 1;
        int n = nums.size();
        for (int i=0;i<n;i++) {
            if (group_sum+nums[i] <=T) {
                group_sum += nums[i];
            }else {
                if (group_sum==0) return false; //要求非空
                group_sum = nums[i];
                if (group_sum > T) return false; // 要求小于上界
                group_cnt++;
            }
        }
        return group_cnt <= m;
    }

   
};
