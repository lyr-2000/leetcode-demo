给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。

回文数
是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

例如，121 是回文，而 123 不是。

class Solution {
public:
    bool isPalindrome(int x) {
        if(x< 0 ) return false;
        int bak = x;
        int j=0;
        int b = (INT_MAX - INT_MAX % 10) / 10;
        while(x) {
            //X MUST >0
            if (j > INT_MAX/10)
                return false;
            if (j == b && x % 10 > INT_MIN % 10  )
                return false;
            j = j*10 + x%10;
            x/=10;
            
        }
        return bak == j;
    }
};

请你来实现一个 myAtoi(string s) 函数，使其能将字符串转换成一个 32 位有符号整数。

函数 myAtoi(string s) 的算法如下：

空格：读入字符串并丢弃无用的前导空格（" “）
符号：检查下一个字符（假设还未到字符末尾）为 ‘-’ 还是 ‘+’。如果两者都不存在，则假定结果为正。
转换：通过跳过前置零来读取该整数，直到遇到非数字字符或到达字符串的结尾。如果没有读取数字，则结果为0。
舍入：如果整数数超过 32 位有符号整数范围 [−231, 231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被舍入为 −231 ，大于 231 − 1 的整数应该被舍入为 231 − 1 。
返回整数作为最终结果。

class Solution {
public:
    int myAtoi(string s) {
        int k = 0;
        int n = s.size();
        bool flag = false;
        int state = 0;

        // 跳过前导空格
        int start = 0;
        while (start < n && s[start] == ' ') start++;

        for (int i = start; i < n; i++) {
            if (state > 0 && !(s[i] >= '0' && s[i] <= '9')) {
                return k;
            }

            if (s[i] == ' ') {
                return k;
            }

            if (s[i] == '+') {
                if (state != 0) return k;  // 遇到非首字符的符号
                state = 1;
                continue;
            }

            if (s[i] == '-') {
                if (state != 0) return k;  // 遇到非首字符的符号
                flag = true;
                state = 1;
                continue;
            }

            if (s[i] >= '0' && s[i] <= '9') {
                state = 1;
                int ne = s[i] - '0';
                if (flag) ne *= -1;

                // 检查溢出条件
                if (k > INT_MAX / 10 || (k == INT_MAX / 10 && ne > INT_MAX % 10 )) { // ne > 8
                    return INT_MAX;
                }
                if (k < INT_MIN / 10 || (k == INT_MIN / 10 && ne < INT_MIN % 10)) { // ne < -7
                    return INT_MIN;
                }

                k = k * 10 + ne;
            } else {
                break;  // 遇到非数字字符直接退出
            }
        }

        return k;
    }
};

给你一个 32 位的有符号整数 x ，返回将 x 中的数字部分反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围 [−231, 231 − 1] ，就返回 0。

假设环境不允许存储 64 位整数（有符号或无符号）。

class Solution {
public:
    int reverse(int x) {
        int j = 0;

        while (x != 0) {
            int digit = x % 10;

            // 检查是否会溢出
            if (j > INT_MAX / 10 || (j == INT_MAX / 10 && digit > INT_MAX %10 )) {
                return 0; // 正溢出
            }
            if (j < INT_MIN / 10 || (j == INT_MIN / 10 && digit < INT_MIN %10)) {
                return 0; // 负溢出
            }

            j = j * 10 + digit;
            x /= 10;
        }

        return j;
    }
};
