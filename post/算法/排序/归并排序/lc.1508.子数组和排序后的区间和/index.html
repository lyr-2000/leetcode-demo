1508. 子数组和排序后的区间和

Description

Difficulty: 中等

Related Topics: 数组 , 双指针 , 二分查找 , 排序

给你一个数组 nums ，它包含 n 个正整数。你需要计算所有非空连续子数组的和，并将它们按升序排序，得到一个新的包含 n * (n + 1) / 2 个数字的数组。

请你返回在新数组中下标为left 到 right **（下标从 1 开始）**的所有数字和（包括左右端点）。由于答案可能很大，请你将它对 10^9 + 7 取模后返回。

示例 1：

1
2
3

输入：nums = [1,2,3,4], n = 4, left = 1, right = 5
输出：13 
解释：所有的子数组和为 1, 3, 6, 10, 2, 5, 9, 3, 7, 4 。将它们升序排序后，我们得到新的数组 [1, 2, 3, 3, 4, 5, 6, 7, 9, 10] 。下标从 le = 1 到 ri = 5 的和为 1 + 2 + 3 + 3 + 4 = 13 。

示例 2：

1
2
3

输入：nums = [1,2,3,4], n = 4, left = 3, right = 4
输出：6
解释：给定数组与示例 1 一样，所以新数组为 [1, 2, 3, 3, 4, 5, 6, 7, 9, 10] 。下标从 le = 3 到 ri = 4 的和为 3 + 3 = 6 。

示例 3：

输入：nums = [1,2,3,4], n = 4, left = 1, right = 10
输出：50

提示：

  • 1 <= nums.length <= 10^3
  • nums.length == n
  • 1 <= nums[i] <= 100
  • 1 <= left <= right <= n * (n + 1) / 2

class Solution {
public:

struct Data {
    int i;
    int j;
    int sum;
    bool operator <(const Data &b) const {
        return sum > b.sum;
    }
};



    int rangeSum(vector<int>& nums, int n, int left, int right) {
        priority_queue<Data> q;
        for (int i=0;i<n;i++) 
            q.push({i,i,nums[i]});  //多路归并排序
        int ans=0,mod = 1e9+7;
        for (int i=1;i<=right;i++) {
            Data d = q.top();
            q.pop();
            if (i>=left)  ans = (ans + d.sum) % mod; //求和
            if (d.j+1 < n) {
                q.push(Data{d.i,d.j+1,(d.sum + nums[d.j+1]) % mod});
            }
        }
        return ans;
    }
};
