单调队列解法

  • 区间[l,r] 可以由 [1,r] 和 [1,l-1] 的信息导出
  • 满足区间减法，可以用前缀和

维护信息是一个点的话，可以考虑使用双指针，如果是一个段的话，可以考虑前缀和和单调队列

如果 区间的两个前缀作差，可以知道 [l,r] 之间的信息，就考虑用前缀和，否则就要考虑其他算法了

通用问题 RMQ问题

问题引入：

RMQ(x,y) 就是询问数组 [x,y] 区间内部的最小值,
现在如果固定区间的尾部，例如 rmq(x,7), 请问 如下序列中最少记录几个元素，满足 rmq(x,7)满足任何需求

比如滑动窗口的最小值，我们可以用一个单调递增队列(双端队列), 头部访问 o(1) 的时间复杂度获取区间最小元素

    q.push(arr[i]);
    // k = y-x
    if(i - q.front() < k )  continue;
    q.pop_front(); //清除过期元素
    cout << arr[q.front()] <<endl; // 滑动窗口的最小值
}

单调队列的性质

入队操作

队尾入队，会把之前破坏单调性的元素从队尾移除（维护单调性）

出队操作

队列头部元素超出区间范围，就移除队头
