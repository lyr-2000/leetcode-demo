[940] 不同的子序列 II

https://leetcode.cn/problems/distinct-subsequences-ii/description/

algorithms
Hard (53.10%)
Likes: 328
Dislikes: 0
Total Accepted: 31.3K
Total Submissions: 59K
Testcase Example: ‘“abc”’

给定一个字符串 s，计算 s 的 不同非空子序列 的个数。因为结果可能很大，所以返回答案需要对 10^9 + 7 取余 。

字符串的 子序列 是经由原字符串删除一些（也可能不删除）字符但不改变剩余字符相对位置的一个新字符串。

例如，“ace” 是 “abcde” 的一个子序列，但 “aec” 不是。

示例 1：

输入：s = “abc”
输出：7
解释：7 个不同的子序列分别是 “a”, “b”, “c”, “ab”, “ac”, “bc”, 以及 “abc”。

示例 2：

输入：s = “aba”
输出：6
解释：6 个不同的子序列分别是 “a”, “b”, “ab”, “ba”, “aa” 以及 “aba”。

示例 3：

输入：s = “aaa”
输出：3
解释：3 个不同的子序列分别是 “a”, “aa” 以及 “aaa”。

提示：

1 <= s.length <= 2000
s 仅由小写英文字母组成

解题思路

参考

根据题意我们设dp[i]为前i个字符可以组成的不同的子序列，则有

$$
dp[i] = dp[i - 1] + newCount - repeatCount
$$

// @lc code=start
class Solution {
   long long M = 1e9 + 7;
public:
   int distinctSubseqII(string s) {
       #define ll long long
       int n = s.size();
       vector<ll> m(26,0);
       ll su = 0;
       for (int i=0;i<n;i++) {
           ll t = m[s[i] - 'a'];
           // 以 s[i] 结尾的个数
           m[s[i] - 'a'] = su + 1;
           su += (su +1 - t) % M;
       }
       return su % M;

   }
};
// @lc code=end

int main() {
   cout << " * [940] 不同的子序列 II" << endl;
   Solution k;
   print( k.distinctSubseqII("aaa"));
   print( k.distinctSubseqII("aba"));
   print( k.distinctSubseqII("abc"));

   return 0;
}

class Solution {
public:
    /*
    abcbb
    a:	""
    	a
    ab:	"",	a
    	b,	ab
    abc: "", a, b, ab
    	 c,	 ac,bc,abc
    abcb:"", a, b, ab, c, ac, bc, abc
    	(b, ab),bb,abb,cb,acb,bcb,abcb		//重复部分是上一次新增字符b的新增部分
    abcbb:"", a, b, ab, c, ac, bc, abc, b, ab, bb, abb, cb, acb, bcb, abcb
    	  (b,ab,bb,abb,cb,acb,bcb,abcb),bb,abb,bbb,abbb,cbb,acbb,bcbb,abcbb 
    	  //重复部分是上一次新增字符b的新增部分 = 第一次新增b的净新增部分 b,ab 加上 第二次新增b的净新增部分 bb,abb,cb,acb,bcb,abcb
    */
    int distinctSubseqII(string s) {
        //为了记录上一次增加相同字符时的新增量
        vector<int> pre(26);
        //初始条件：0个字符时，有1个子序列（空字符串）
        int newCount, ans = 1, mod = 1e9 + 7;

        for(char chs : s)
        {
            //新增子序列（不考虑重复的部分）
            newCount = ans;
            //dp[i + 1] = dp[i] + 新增子序列（不考虑重复的部分） - 重复部分（上次增加相同字符时的所有新增部分（每次新增相同字符的净新增的总和）） 
            ans = ((ans << 1) % mod - pre[chs - 'a'] % mod + mod) % mod;
            //更新该字符上次新增的部分
            pre[chs - 'a'] = newCount;     
        }

        return ((ans - 1 + mod) % mod);
    }
};
