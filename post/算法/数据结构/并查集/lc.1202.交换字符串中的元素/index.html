1202. 交换字符串中的元素

Description

Difficulty: 中等

Related Topics: 深度优先搜索 , 广度优先搜索 , 并查集 , 哈希表 , 字符串

给你一个字符串 s，以及该字符串中的一些「索引对」数组 pairs，其中 pairs[i] = [a, b] 表示字符串中的两个索引（编号从 0 开始）。

你可以 任意多次交换 在 pairs 中任意一对索引处的字符。

返回在经过若干次交换后，s 可以变成的按字典序最小的字符串。

示例 1:

输入：s = "dcab", pairs = [[0,3],[1,2]]
输出："bacd"
解释： 
交换 s[0] 和 s[3], s = "bcad"
交换 s[1] 和 s[2], s = "bacd"

交换 s[0] 和 s[2], s = "acbd"

提示：

  • 1 <= s.length <= 10^5
  • 0 <= pairs.length <= 10^5
  • 0 <= pairs[i][0], pairs[i][1] < s.length
  • s 中只含有小写英文字母

Solution

Language: C++

解题思路

假设 0,1 可以互换，1,3可以互换，那么 0,1,3 都是连通的，可以互换，题目说可以任意次互换，于是考虑并查集加小顶堆，小顶堆对字符排序就可以了。

class Solution {
public:

    string smallestStringWithSwaps(string s, vector<vector<int>>& pairs) {
        // string temp = p;
        int n = s.size();
        int m = pairs.size();
        init(n);
        priority_queue<char,vector<char>,greater<char> > q[n];// 对字符排序
        for (int i=0;i<m;i++)  {
            auto temp = pairs[i];
            merge(temp[0],temp[1]);
        }
        for (int i=0;i<n;i++) {
            q[find(i)].push(s[i]);
        }
        string res;
        for (int i=0;i<n;i++) {
            res.push_back(q[find(i)].top());
            q[find(i)].pop();
        }
        return res;
    }
    vector<int> p;
    void init(int n) {
        p.resize(n);
        for (int i=0;i<n;i++) 
            p[i] = i;
    }
    int find(int x) {
        if (x!=p[x])
            p[x] = find(p[x]);
        return p[x];
    }
    void merge(int a,int b) {
        p[find(a)]  = find(b);
    }
};
