前缀和数组算法理解

lowbit 函数

lowbit(i): 代表i这个数字， 二进制表示的最后一位1的位权：

$lowbit(8) = (1000)_2 = 8$

$lowbit(x) = x&(-x)$

改进前缀和数组

tree[i] 代表 C[i] 代表前lowbit(i) 项的和

原来前缀和数组的定义是 [1,n] 项的 数组和，由于 sum[n] 与 {1,n} 项都有关系，就导致了单点修改的问题比较大， 所以前缀和数组改进的方式在于 让 sum[n] 只记录 前 lowbit(n) 项的和。

如图所示
$$
lowbit(8) = 8\
lowbit(7)=1\
lowbit(6) = 2\

$$

如图所示，用树状数组的话修改 nums[1] ,只需要修改 C[1],C[2],C[4],和 C[8] 的值，不需要 修改 C[1~n] 的所有值

树状数组求和公式

$S[i] = S[i-lowbit(i)] + C[i]$

例如
$$
sum[7] = sum[6] + nums[7] \
sum[6] = sum[4] + nums[6]
$$

树状数组单点修改

修改A[5],那么要更新 C[5],C[6],C[8] 的值

 #define lowbit(i) (i & (-i))

// lowbit 用来求二进制的最后1位1
// 负数= 补码= 反码+1
// 计算机中没有减法，所以用补码表示法可以将减法转化为加法，所以能够成功
class FenwickTree
{
public:
    FenwickTree(int n) : n(n), c(n + 1) {}

    void add(int i, int x)
    {
        //原数组的 第 i位 加上某个值x
        while (i <= n)
        {
            c[i] += x;
            i += lowbit(i);
        }
    }
    int query_sum(int x)
    {
        int sum = 0;
        while (x)
        {
            sum += c[x];
            x -= lowbit(x);
        }
        return sum;
    }
    void output()
    {
        for (int i = 1; i <= n; i++)
        {
            cout << c[i] << " ";
        }
        cout << endl;
        return;
    }

private:
    int n;
    vector<int> c;
};

int main()
{
    int n;
    cin >> n;
    FenwickTree tree(n);
    for (int i = 1; i <= n; i++)
    {
        int k;
        cin >> k;
        tree.add(i, k);
    }
    tree.output();
}

差分原理和应用

原数组的区间操作，等价于 差分数组的单点操作

a[l~r] 区间都 +=x, 那么等价于 d[l] += x, d[r+1]-=x

通过这个性质，我们可以把区间加和操作，转化为差分数组的2次单点修改操作
