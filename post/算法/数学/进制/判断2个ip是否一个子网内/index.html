IP地址是由4个0-255之间的整数构成的，用".“符号相连。
二进制的IP地址格式有32位，例如：10000011，01101011，00000011，00011000;每八位用十进制表示就是131.107.3.24
子网掩码是用来判断任意两台计算机的IP地址是否属于同一子网络的根据。
子网掩码与IP地址结构相同，是32位二进制数，由1和0组成，且1和0分别连续，其中网络号部分全为“1”和主机号部分全为“0”。
你可以简单的认为子网掩码是一串连续的1和一串连续的0拼接而成的32位二进制数，左边部分都是1，右边部分都是0。
利用子网掩码可以判断两台主机是否中同一子网中。
若两台主机的IP地址分别与它们的子网掩码进行逻辑“与”运算（按位与/AND）后的结果相同，则说明这两台主机在同一子网中。

** 输入描述： **
3行输入，第1行是输入子网掩码、第2，3行是输入两个ip地址
题目的示例中给出了三组数据，但是在实际提交时，你的程序可以只处理一组数据（3行）。

** 输出描述： **
若IP地址或子网掩码格式非法则输出1，若IP1与IP2属于同一子网络输出0，若IP1与IP2不属于同一子网络输出2

如何校验 mask

  1. 检查mask: 全零与全一不是子网掩码，是否连续的1接连续的0。

其他解法

f 在十六进制中表示的是 15，而 15 转换为二进制就是 1111。这是因为十六进制是基于 16 进制的，每一位可以表示的数字范围是从 0 到 15，而 f 代表的是 15。

为什么 f 是 4 个 1？

十六进制与二进制的关系：

  • 十六进制的每一位可以表示 16 个不同的值，从 0 到 f（即 15）。因此，1 位十六进制数字可以通过 4 位二进制表示。
  • 十六进制的 f 表示的是数字 15，而 15 转换成二进制就是 1111。

计算方法：

  • 十进制的 15：

    • 15 / 2 = 7 余 1
    • 7 / 2 = 3 余 1
    • 3 / 2 = 1 余 1
    • 1 / 2 = 0 余 1

    反向排列余数：1111（二进制表示）。

  • 在十六进制中，f 对应的十进制值是 15，它的二进制表示就是 1111，因此一个十六进制的 f 就等于四个二进制的 1。

    }

package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func parse(s string) (map[int]int, bool) {
	var ip = make(map[int]int, 4)
	all := strings.FieldsFunc(s, func(r rune) bool { return r == '.' })
	if len(all) != 4 {
		return ip, false
	}
	for i, v := range all {
		b, err := strconv.Atoi(v)
		if err != nil {
			return ip, false
		}
		ip[i+1] = b
		if b < 0 || b > 255 {
			return ip, false
		}
	}
	return ip, true
}

// 注意： 是要 uint32, 如果64 是有问题的
func ismask(ip map[int]int) bool {
	var w uint32
	for i := 1; i <= 4; i++ {
		w = w<<8 + uint32(ip[i])
	}
	m := w
	if m == 0 || m == 0xffffffff {
		return false
	}

	for m&0x80000000 != 0 {
		m = m << 1
	}

	return m == 0
}
func ismaskx(ip map[int]int) bool {
	for i := 1; i < 4; i++ {
		if ip[i] < 255 && ip[i+1] > 0 { //网络位：1 主机位0,
			return false
		}
	}
	return true
}
